Resumen de una variable continua

  • summary() : resumen estad√≠stico de r-base

  • Las funciones de resumen estad√≠stico t√≠pico con R son :

  • mean() : calcula la media

  • sd() : calcula la desviaci√≥n est√°ndar

  • min() : valor m√≠nimo

  • max() : valor m√°ximo

  • median() : calcula la mediana

  • length() : largo o n√∫mero de valores

  • range() : calcula el valor m√≠nimo y m√°ximo

  • quantile() : calcula los percentiles de un vector num√©rico`
